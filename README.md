# GLaDOS

## [Documentation](https://kenan-blasius.github.io/glados-doc/)

## Description

GLaDOS est un projet de création de langage de programmation.

## Installation

Clonez le repo et lancez `make re`.

## Utilisation

Lancez `./glados <fichier>` pour compiler un fichier.
